Data and Code for Mbiol Thesis: 
----------
This repository contains all the data and code for the phylogenies and statistical analysis presented in my Master's thesis. 

**Data:**
-----------
Supplementary tables S1-S7 are described in the Supplementary Information section of my thesis, and can be downloaded here.

**Code:**
-------------
There are two folders containing code. Each folder contains a quarto file with annotated code, and the associated data files:
1. **analysis_pipeline**
   - Contains a record of code used in the bioinformatic pipeline for sequence extraction, alignment and phylogeny generation. 

2. **phylogeny_visualisation**
   - R code used to visualise the phylogenies and generate the figures used in my thesis.

The whole folder should be downloaded and the working directory set to here in order to run the code.
